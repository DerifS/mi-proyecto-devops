# Proyecto DevOps

## Descripción del proyecto
Este repositorio contiene un sitio web básico en HTML y CSS, acompañado de un script de shell (`deploy.sh`) que simula el despliegue a producción. Todo el proyecto está diseñado para funcionar en GitHub como plataforma principal de DevOps y control de versiones.

## Objetivo
Poner en práctica y demostrar el dominio de conceptos fundamentales de control de versiones con Git, así como la integración de herramientas y flujos de trabajo DevOps básicos utilizando GitHub.

## Estructura del repositorio
El proyecto sigue una organización clara, separando el código fuente de los scripts operativos:

```text
